# Spectral-Spooky-Ghost-3JS

An immersive 3D web experience featuring a spectral ghost that interacts with your mouse movements. Built with Three.js, this project showcases advanced visual effects including bloom lighting, analog decay shaders, particle systems, and atmospheric rendering.

## Features

- **Interactive Ghost**: Follows mouse cursor with smooth animations and wobbling effects
- **Glowing Eyes**: Eyes light up based on movement with customizable glow colors
- **Particle System**: Leaves trailing particles when moving, with various shapes and colors
- **Fireflies**: Ambient glowing fireflies that float around the scene
- **Post-Processing Effects**:
  - Bloom lighting for ethereal glow
  - Analog decay shader simulating VHS/VCR effects (grain, bleeding, scanlines, jitter)
  - Atmosphere reveal effect that uncovers the scene around the ghost
- **Real-time GUI**: Tweak all parameters live using Tweakpane controls
- **Responsive Design**: Adapts to window resizing

## Technologies Used

- **Three.js**: 3D graphics and rendering
- **WebGL**: Hardware-accelerated graphics
- **Tweakpane**: Real-time parameter tweaking
- **ES Modules**: Modern JavaScript imports
- **Post-Processing**: Advanced shader effects

## How to Run

1. Clone the repository
2. Open `index.html` in a modern web browser
3. Move your mouse to interact with the ghost
4. Use the GUI panel in the top-right to adjust effects

No build process or dependencies required - everything loads from CDN.

## Browser Compatibility

Requires a modern browser with WebGL support:
- Chrome 51+
- Firefox 51+
- Safari 10+
- Edge 79+
